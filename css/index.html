关于

计科七班成立于2015年9月初旬，在2016年8月份进行人员变动，目前本班由46名男生和3名女生组成。 其中两名休学，两名当兵。在班长等班委的带领下，计科七班正在茁壮成长，相信在不久的将来，我们将会成为一个优秀并且团结的大家庭。
